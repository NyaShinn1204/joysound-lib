## Installation

**※ Python 3.10 以上のバージョンが必要です。**

「joysound-lib」をインストールするには、以下のコマンドを実行します:

> [!TIP]
> 開発バージョンです! 注意をしてください

```bash
git clone https://github.com/nyashinn1204/joysound-lib.git

cd joysound-lib

make setup

make active
```
